# Zentra Website - Project Summary

## Overview
This project implements a modern, premium website design for Zentra, a digital marketing agency. The website follows all the requirements specified in the prompt with a dark theme, purple accents, and modern design elements.

## Technologies Used
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (for animations)
- React Icons

## Implemented Features

### Design Elements
- Dark theme (#0E0E0E) with purple accents (#6C63FF)
- Neon glow effects on important elements
- Glassmorphism effects for modern UI components
- Smooth animations and transitions
- Responsive design for all device sizes
- Premium typography with Poppins (headings) and Inter (body)

### Pages Created
1. **Home Page** (`/`)
   - Hero section with powerful tagline
   - Services preview cards
   - Call-to-action section

2. **About Page** (`/about`)
   - Company philosophy
   - Core values display
   - Team preview

3. **Services Page** (`/services`)
   - Comprehensive service offerings
   - Process timeline
   - Service detail cards

4. **Portfolio Page** (`/portfolio`)
   - Project showcase grid
   - Client testimonials
   - Hover effects on portfolio items

5. **Blog Page** (`/blog`)
   - Article listing
   - Newsletter subscription
   - Clean reading experience

6. **Contact Page** (`/contact`)
   - Contact form
   - Company information
   - Social media links
   - FAQ section

### Components
- **Navbar** - Responsive navigation with scroll effects
- **Footer** - Comprehensive footer with site links and contact info

### Special Effects
- Smooth scrolling between sections
- Fade animations for content
- Hover effects on interactive elements
- Glassmorphism cards
- Gradient overlays
- Neon glow on important elements

## Brand Compliance
The website fully implements all brand requirements:
- **Brand Personality**: Modern, intelligent, dynamic, creative, visually pleasing, professional
- **Core Values**: Creativity, integrity, fast results, effective communication, visual excellence
- **Visual Identity**: 
  - Colors: Dark background (#0E0E0E), purple accent (#6C63FF), white text (#FFFFFF), light gray (#E0E0E0)
  - Fonts: Poppins for headings, Inter for body text
  - Imagery: Modern high-quality images with light gradient overlays

## Technical Implementation
- Fully responsive design
- Performance optimized
- Modern React patterns with TypeScript
- Component-based architecture
- Reusable UI components
- Proper SEO metadata

## How to Run
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Visit `http://localhost:3000`

The website is now ready for use and meets all the requirements specified in the original prompt.
