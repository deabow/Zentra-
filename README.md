# زنترا - موقع وكالة التسويق الرقمي

موقع عصري ومُبسط لزنترا، وكالة تسويق رقمي متكاملة.

## الميزات

- **تصميم نظيف**: سمة داكنة مع لهجة بنفسجية
- **متجاوب بالكامل**: يعمل على جميع أحجام الأجهزة
- **رسوم متحركة خفيفة**: تجربة مستخدم محسّنة دون تأثيرات مفرطة
- **تأثيرات زجاجية**: عناصر واجهة مستخدم عصرية مع ضبابية خلفية خفيفة
- **Next.js 14**: مبني بأحدث App Router
- **Tailwind CSS**: إطار عمل CSS يعتمد على الأدوات
- **TypeScript**: JavaScript آمن من حيث النوع

## الصفحات

1. **الرئيسية** - الصفحة الرئيسية مع قسم البطل ومعاينة الخدمات
2. **عنّا** - معلومات الشركة والقيم
3. **الخدمات** - تفصيل شامل لجميع الخدمات المقدمة
4. **الأعمال** - عرض الأعمال السابقة
5. **المدونة** - رؤى تسويقية واتجاهات صناعية
6. **تواصل معنا** - نموذج للتواصل ومعلومات الاتصال

## البدء

أولاً، قم بتثبيت الاعتمادات:

```bash
npm install
```

ثم، قم بتشغيل خادم التطوير:

```bash
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) باستخدام متصفحك لرؤية النتيجة.
